## [0.1.0] - 2026-03-05

### Added
- Initial release of WildEdge Python SDK
- Support for ONNX Runtime, GGUF/llama.cpp, timm, PyTorch, and Keras integrations
- Automatic latency and error tracking without capturing inputs/outputs
- Event batching and transmission to WildEdge servers
- Configuration via DSN and environment variables
- Comprehensive test suite with tox support for multiple Python versions
- CI/CD pipeline with GitHub Actions
